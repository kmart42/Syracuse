To find the contiguous subsequence with the greatest sum, a bottom up Dynamic Programming 
algorithm works:\\\\
def maxArray(c[], size)\\
\tab total = c[0]\\
\tab current = c[0]\\
\tab for 1 to size:\\
\tab \tab current = max(c[i], current + c[i])\\
\tab \tab total = max(total, current)\\
\tab return total
